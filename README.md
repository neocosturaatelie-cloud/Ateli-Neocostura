# 🪡 Ateliê Neocostura - Site Oficial

Site institucional elegante e moderno para o **Ateliê Neocostura**, especializado em ajustes, reformas, customização, sob medidas, bordados computadorizados e fantasias.

## 🎨 Sobre o Projeto

Site desenvolvido com design sofisticado, feminino e profissional, utilizando a paleta de cores pink (#D81B60) e creme (#F5ECD7). O site apresenta todos os serviços do ateliê de forma elegante e acessível.

## ✨ Funcionalidades Implementadas

### 📱 Seções do Site
- **Hero Section**: Banner principal com título impactante e call-to-actions
- **Sobre**: Apresentação do ateliê com features destacadas
- **Serviços**: 6 cards apresentando os serviços oferecidos
  - Ajustes
  - Reformas
  - Customização
  - Sob Medidas
  - Bordados Computadorizados
  - Fantasias
- **Galeria**: Espaço preparado para fotos dos trabalhos
- **Contato**: Informações completas de localização e contato
- **Footer**: Rodapé com redes sociais e informações

### 🎯 Recursos Técnicos
- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Menu de navegação fixo com efeito ao scroll
- ✅ Menu mobile com animação hamburger
- ✅ Smooth scroll entre seções
- ✅ Animações sutis ao fazer scroll (fade in)
- ✅ Integração com WhatsApp
- ✅ Link para Instagram
- ✅ Mapa do Google Maps integrado
- ✅ Ícones do Font Awesome
- ✅ Tipografia elegante (Playfair Display + Poppins)
